Help
  • Encode will assign an Emoji to every character you type.
  • Translate will replace any word you type with a corresponding Emoji.
  • Madlib will replace all category words with a randomized Emoji of that category.
  • Search will search the Emoji database for a match with the text that you type.
  • Random will randomly select one of the previous four translation options for your text.
